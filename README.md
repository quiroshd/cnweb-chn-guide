# cnweb-chn-guide

本仓库用于归档与发布多个独立的 HTML 页面。这些页面内容涉及中文网络环境下的各类信息与资源整理，不针对任何单一域名或具体网站。

## 项目简介

该仓库的主要目的是提供一个结构化的空间，用于存放和展示多个独立的 HTML 文档。这些文档可能包含指南、列表、说明或其他静态内容，所有文件均以纯前端形式呈现，无需后端支持。

## 目录说明

- `pages/` — 存放所有独立的 HTML 页面文件，每个文件对应一个独立的主题或内容块。
- `assets/` — 存放页面所需的静态资源，例如样式表、图片或脚本（如有）。
- `docs/` — 可选目录，用于存放与项目相关的说明文档或引用资料。

## 页面归档说明

每个 HTML 页面均为独立完整的文档，不依赖其他页面或外部服务。页面内容可能随时间更新，但历史版本不会主动删除。建议使用者根据自身需求，直接访问或引用对应页面。

如需查看所有可用页面，请浏览 `pages/` 目录下的文件列表。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时会通过 commit 记录说明变更内容。
- 不保证所有页面始终可用或内容完全准确，使用者请自行判断。
- 欢迎通过 Issues 提出建议或报告问题，但请勿提交包含具体营销链接或诱导点击的内容。
- Pull Request 仅接受与归档目的相符的独立 HTML 页面提交。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可查看内容。
3. 也可通过 GitHub Pages 或其他静态托管服务部署整个仓库，以便在线访问。
